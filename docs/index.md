# Portal Documentation

Portal is a local-first file transfer tool that started as a personal way to move files from phone to computer. The goal is to make transfers effortless:

- CLI ↔ CLI (today)
- CLI ↔ Browser (planned)
- Browser ↔ CLI (planned)
